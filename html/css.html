¿Qué es CSS?

CSS son las siglas de Cascading Style Sheets o Hojas de Estilo en Cascada. Es un lenguaje que se utiliza para describir la presentación de documentos escritos en lenguajes de marcado como HTML o XML.

CSS en línea

este es un claro ejemplo: <p style="color:red">Texto en rojo</p>

CSS interno

<style> p {color:blue;} </style>

CSS externo

Se usa un archivo separado enlazado con <link>.
