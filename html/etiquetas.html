1. Etiquetas de Documento

Las etiquetas de documento definen la estructura básica sobre la cual se construye cualquier página web. Estas etiquetas permiten que el navegador interprete correctamente el contenido y lo presente al usuario.

1.1 <html>

La etiqueta <html> es la raíz del documento y encierra todo el contenido HTML. Incluye el atributo lang, que define el idioma principal de la página, lo cual es fundamental para la accesibilidad y el posicionamiento en buscadores.

1.2 <head>

El <head> almacena información esencial que no es visible al usuario, como configuraciones, enlaces externos, metadatos y scripts necesarios antes de cargar la página. Afecta el funcionamiento general del sitio.

1.3 <body>

Dentro del <body> se encuentra todo el contenido visible por el usuario: textos, imágenes, botones, formularios, etc. El navegador interpreta esta sección y la convierte en la interfaz visual del sitio web.

2. Etiquetas de Meta Datos

Estas etiquetas contienen información que ayuda a los navegadores y motores de búsqueda a interpretar correctamente la página.

2.1 <meta>

La etiqueta <meta> define metadatos como codificación, descripción, autor o palabras clave. Aunque no es visible, contribuye al SEO y al comportamiento del navegador.

2.2 <style>

<style> permite insertar reglas CSS directamente dentro del documento. Se usa para dar estilo, aunque normalmente se recomienda usar archivos CSS externos para mejor organización.

2.3 <title>

La etiqueta <title> define el título que aparece en la pestaña del navegador y en los resultados de búsqueda. Es clave para la identidad y posicionamiento del sitio.

2.4 <base>

<base> establece una URL base que se aplica a todos los enlaces relativos del documento. Es útil en sitios con estructuras complejas o múltiples rutas.

2.5 <link>

<link> conecta el documento con recursos externos como hojas de estilo, fuentes o iconos. Es fundamental para definir el estilo y apariencia del sitio.

3. Etiquetas de Agrupación

Sirven para organizar el contenido dentro de la página. Estas etiquetas permiten dividir el documento en secciones manejables y estructuradas.

3.1 <p>

La etiqueta <p> define un párrafo. Facilita la lectura y separación del contenido textual.

3.2 <hr>

<hr> crea una línea horizontal que sirve como separador visual entre secciones o ideas.

3.3 <ul>

<ul> representa una lista no ordenada. Se usa para mostrar información en forma de viñetas.

.4 <div>

<div> es un contenedor genérico que sirve para agrupar elementos. Es fundamental en el diseño y estructura de la página.

3.5 <figure>

<figure> encapsula contenido visual como imágenes o gráficos, y suele ir acompañado de <figcaption>.

3.6 <figcaption>

<figcaption> proporciona una descripción o pie de imagen dentro de una etiqueta <figure>.

4. Etiquetas Textuales

Estas etiquetas modifican y estructuran el contenido escrito dentro del documento.

4.1 <a>

<a> crea enlaces a páginas internas, externas, descargas o secciones específicas dentro de la misma página.

4.2 <span>

<span> es un contenedor en línea usado para resaltar, modificar o aplicar estilos a partes específicas del texto.

4.3 <sub>

<sub> muestra texto en subíndice, útil en fórmulas químicas como H₂O.

4.4 <sup>

<sup> coloca el texto en superíndice, como en potencias o números ordinales (x², 1º).

4.5 <cite>

<cite> identifica el título de una obra, libro o referencia, útil en textos académicos.

5. Etiquetas Semánticas

Estas etiquetas agregan significado al contenido y mejoran la estructura lógica de la página.

5.1 <nav>

<nav> define una sección de navegación con enlaces a otras partes del sitio.

5.2 <section>

<section> representa una sección temática dentro del documento.

5.3 <article>

<article> representa contenido independiente como artículos, posts o noticias.

5.4 <header>

<header> define encabezados que pueden contener títulos, logos o menús.

5.5 <main>

<main> contiene el contenido principal de la página. Solo puede existir uno por documento.

5.6 <footer>

<footer> representa el pie de página, donde suelen colocarse enlaces secundarios, créditos e información legal.
