    • Atributos globales

    Los atributos globales son aquellos que pueden ser utilizados en cualquier elemento HTML, a excepción de las etiquetas que no están incluidas en el cuerpo del documento (como <html>, <head>, <title>). Proporcionan información básica o comportamiento estándar.

    1.1.

    Estos son los atributos globales más utilizados, fundamentales para la manipulación de estilos (CSS) y la interactividad (JavaScript).

    id (Identificador Único):
    Asigna un identificador único al elemento. Regla: Solo debe haber un elemento con un id específico por página.
    Uso: Se utiliza para apuntar a un elemento específico desde CSS (usando el selector #) o desde JavaScript (usando document.getElementById()).


    class (Clases de Estilo):
    Asigna uno o más nombres de clase al elemento.
    Regla: Puede haber múltiples elementos con la misma clase.
    Uso: Se utiliza para aplicar estilos CSS a un grupo de elementos (usando el selector .) o para seleccionar un grupo de elementos en JavaScript.
